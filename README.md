# 英文打字練習 | English Typing Practice

一個功能完整的免費線上英文打字練習工具，提供 B1-C1 級別的英語句子練習，幫助提升英文打字速度與準確率。具備 PWA 功能、完整無障礙支援、效能監控等企業級特性。

🌐 **線上體驗**: https://ben0126.github.io/english_typing_practice/

## ✨ 主要功能

### 🎯 多級別練習
- **B1 中級**: 適合有一定英語基礎的學習者
- **B2 中高級**: 適合中高級英語學習者
- **C1 高級**: 適合高級英語學習者

### 🔊 語音功能
- **句子播放**: 點擊播放完整英文句子
- **慢速播放**: 提供較慢語速的句子播放
- **語速控制**: 可調整播放語速 (0.3x - 1.5x)
- **單字發音**: 點擊單字或發音按鈕聽取單字發音

### 📊 即時統計
- **WPM 計算**: 即時顯示每分鐘字數
- **準確率統計**: 即時計算打字準確率
- **進度追蹤**: 顯示當前練習進度

### 🌏 雙語學習
- **中文翻譯**: 每個英文句子都配有中文翻譯
- **重點單字**: 標註重要單字並提供中文解釋
- **詞彙學習**: 邊練習打字邊擴充英語詞彙量

### 💻 跨平台支援
- **響應式設計**: 支援電腦、平板、手機
- **PWA 功能**: 可安裝到桌面和手機
- **離線使用**: 網路斷線時仍可使用
- **完全免費**: 無需註冊或付費

## 🎮 使用方式

1. **選擇難度**: 點擊 B1、B2 或 C1 按鈕選擇適合的級別
2. **聽取發音**: 使用播放按鈕聽取句子或單字發音
3. **開始打字**: 在輸入框中輸入上方顯示的英文句子
4. **查看統計**: 即時查看 WPM 和準確率
5. **學習單字**: 查看重點單字和中文翻譯
6. **下一句**: 完成後點擊「下一句」繼續練習

## ⌨️ 鍵盤快捷鍵

### 基本操作
- `Ctrl + Enter`: 進入下一句練習
- `Ctrl + P`: 播放句子
- `Ctrl + S`: 慢速播放
- `Escape`: 回到輸入框

### 級別切換
- `Alt + 1`: 切換到 B1 中級
- `Alt + 2`: 切換到 B2 中高級
- `Alt + 3`: 切換到 C1 高級

### 導航
- `Tab`: 下一個可聚焦元素
- `Shift + Tab`: 上一個可聚焦元素
- `Space`: 在音頻按鈕上時播放/暫停
