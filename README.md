# 英文打字練習 | English Typing Practice

一個功能完整的免費線上英文打字練習工具，提供 A2-C1 級別的英語句子練習，幫助提升英文打字速度與準確率。具備 PWA 功能、完整無障礙支援、效能監控等企業級特性。

*最後更新：OCT 02 2025*

🌐 **線上體驗**: https://ben0126.github.io/english_typing_practice/

## ✨ 主要功能

### 🎯 雙模式練習系統
- **聽力練習模式**: 聽句子並將聽到的內容打出來，適合提升聽力理解能力
- **純打字練習模式**: 看著英文句子進行打字練習，適合提升打字速度和準確率
- **模式優先選擇**: 使用者需先選擇練習模式，再選擇難度級別

### 🎯 多級別練習
- **所有難度**: 混合所有級別的句子，適合綜合練習
- **A2 初中級**: 適合初級英語學習者
- **B1 中級**: 適合有一定英語基礎的學習者
- **B2 中高級**: 適合中高級英語學習者
- **C1 高級**: 適合高級英語學習者

### 🔊 語音功能
- **句子播放**: 點擊播放完整英文句子
- **慢速播放**: 提供較慢語速的句子播放
- **語速控制**: 可調整播放語速 (0.3x - 1.5x)，預設為1.0x
- **單字發音**: 點擊單字或發音按鈕聽取單字發音
- **統一控制**: 語速控制位於難度選擇區域，操作更便捷

### 📊 即時統計
- **WPM 計算**: 即時顯示每分鐘字數
- **準確率統計**: 即時計算打字準確率
- **進度追蹤**: 顯示當前練習進度

### 🌏 雙語學習
- **中文翻譯**: 每個英文句子都配有中文翻譯
- **重點單字**: 標註重要單字並提供中文解釋
- **詞彙學習**: 邊練習打字邊擴充英語詞彙量
- **靈活顯示**: 可選擇隱藏中文或英文，適應不同學習需求
- **聽力練習**: 隱藏英文句子，專注於聽力理解練習

### 💻 跨平台支援
- **響應式設計**: 支援電腦、平板、手機
- **PWA 功能**: 可安裝到桌面和手機
- **離線使用**: 網路斷線時仍可使用
- **完全免費**: 無需註冊或付費

## 🎯 學習模式

### 🎧 聽力練習模式
- **隱藏英文句子**：只顯示中文翻譯，專注於聽力理解
- **語音播放**：自動播放英文句子，使用者聽後輸入
- **適合對象**：想要提升聽力能力的學習者
- **使用方式**：選擇「聽力練習模式」後，聽取語音並輸入聽到的內容

### ⌨️ 純打字練習模式
- **隱藏中文翻譯**：只顯示英文句子，專注於英文打字
- **視覺練習**：看著英文句子進行打字練習
- **適合對象**：有一定基礎，想要純英文環境的學習者
- **使用方式**：選擇「純打字練習模式」後，看著英文句子進行打字

### 📝 綜合練習模式
- **顯示所有內容**：英文句子 + 中文翻譯 + 重點單字
- **適合對象**：初學者或需要完整學習體驗的用戶
- **使用方式**：保持預設設定，所有內容都顯示

### 🎛️ 語速控制
- **預設語速**：1.0x（正常語速）
- **調整範圍**：0.3x - 1.5x
- **使用建議**：
  - 初學者：0.7x - 0.9x（較慢語速）
  - 進階者：1.0x - 1.2x（正常到稍快語速）
  - 挑戰者：1.3x - 1.5x（快速語速）

## 🎮 使用方式

1. **選擇模式**: 首先選擇「聽力練習模式」或「純打字練習模式」
2. **選擇難度**: 選擇適合的難度級別（所有難度、A2、B1、B2、C1）
3. **進入練習**: 選擇難度後，難度選擇區域會隱藏，顯示簡潔的練習介面
4. **查看設定**: 在練習區域頂部可看到當前的模式和難度設定
5. **調整語速**: 使用語速滑桿調整播放語速（預設1.0x）
6. **選擇情境**: 使用情境下拉選單選擇特定主題（日常對話、學術、商務、旅遊等）
7. **聽取發音**: 使用播放按鈕聽取句子或單字發音
8. **開始打字**: 根據選擇的模式進行打字練習
9. **查看統計**: 即時查看 WPM 和準確率
10. **學習單字**: 查看重點單字和中文翻譯
11. **下一句**: 完成後點擊「下一句」繼續練習
12. **重新選擇**: 如需更改設定，點擊「🔄 重新選擇」按鈕

## ⌨️ 鍵盤快捷鍵

### 基本操作
- `Ctrl + Enter`: 進入下一句練習
- `Ctrl + P`: 播放句子
- `Ctrl + S`: 慢速播放
- `Escape`: 回到輸入框

### 級別切換
- `Alt + 0`: 切換到所有難度
- `Alt + 1`: 切換到 A2 初中級
- `Alt + 2`: 切換到 B1 中級
- `Alt + 3`: 切換到 B2 中高級
- `Alt + 4`: 切換到 C1 高級

### 顯示控制
- `Ctrl + H`: 切換隱藏中文翻譯
- `Ctrl + E`: 切換隱藏英文句子

### 導航
- `Tab`: 下一個可聚焦元素
- `Shift + Tab`: 上一個可聚焦元素
- `Space`: 在音頻按鈕上時播放/暫停
